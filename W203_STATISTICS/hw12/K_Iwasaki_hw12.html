2. Using diagnostic plots, background knowledge, and statistical tests, assess all 6 assumptions of the CLM. When an assumption is violated, state what response you will take.

  1. Linear population model. This assumption is ok.
  2. Random sampling. There is not much information about how this data is collected.I assume this assumption isnt violated.
  3. No perfect multicolliearity. The plot below doesnt show perfect multicolliearity.
plot(data$length, data$rate)

4. Zero-conditional mean. Look at the Residuals vs.Fitted plot. Smooth line in the chart is around 0. It seems this assumption is not violated.

  1. Homoskedasticity. Look at the Residuals vs.Fitted plot and Scale-location plot. A bit difficult to say if the model shows homoskedasticity or not. To double-check, run a Breaush-Pagan test. p-value is large so we dont reject the null hypothesis that the model has homoskedasticity. Thus this assumption is not violated.
bptest(model1)
